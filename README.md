# 📘 Exam Platform

Платформа для проведения экзаменов и тестирования с загрузкой вопросов из DOCX файлов.

## 🚀 Функционал

- ✅ Загрузка вопросов из DOCX/TXT файлов
- ✅ Случайная выборка 30 вопросов из базы
- ✅ Таймер 60 минут
- ✅ Подсчёт правильных ответов
- ✅ Сохранение прогресса (при перезагрузке)
- ✅ Адаптивный дизайн
- ✅ Работает на GitHub Pages + Render.com

## 🛠 Технологии

### Фронтенд
- HTML5 + CSS3
- Vanilla JavaScript
- GitHub Pages (хостинг)

### Бэкенд
- FastAPI (Python)
- Uvicorn (ASGI сервер)
- Render.com (хостинг)
- python-docx (парсинг DOCX)
